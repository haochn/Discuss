# Discuss
React 构建聊天模块
## 初始版本 v0.1
2017.07.14
- 利用state、props在子组件和父组件传递数据
- 初始化用户输入信息组件 TalkInpt、信息总体组件TalkList、单个用户信息组件Talk

**数据结构**：{id:用户id,content:评论信息}

## 版本 v2.0
2017.07.17

- 将id,content数据储存在localstorage
- 新增删除功能和显示评论时间功能

**数据结构**：[{id:用户1,content:评论信息},{id:用户2,content:评论信息}]
